# ⚙️ ExtensionMachine

**Your browser, enhanced.** A single Chrome extension that replaces a dozen others — privacy protection, page cleanup, productivity tools, and more. Everything off by default. Turn on only what you want.

---

## Installation

ExtensionMachine is not yet on the Chrome Web Store. Installing it takes about 60 seconds:

**1. Download the extension**

Click the green **Code** button at the top of this page → **Download ZIP** → unzip the folder somewhere on your computer (e.g. your Desktop or Documents).

**2. Open Chrome Extensions**

In Chrome, go to: `chrome://extensions`

Or: Chrome menu (⋮) → **Extensions** → **Manage Extensions**

**3. Enable Developer Mode**

Toggle the **Developer mode** switch in the top-right corner of the Extensions page.

**4. Load the extension**

Click **Load unpacked** → navigate to and select the unzipped `chromeextension` folder → click **Select**.

ExtensionMachine will appear in your extensions list. Click the puzzle piece icon (🧩) in your Chrome toolbar to pin it for easy access.

## Features

All features are **off by default**. Open the extension popup and toggle what you want.

### 🔒 Privacy

| Feature | What it does |
|---|---|
| **Ad Blocker** | Blocks requests to 93 ad networks including Google Ads, Criteo, Taboola, and Outbrain |
| **Tracker Blocker** | Blocks 65 analytics and tracking scripts including Google Analytics, Meta Pixel, Hotjar, Mixpanel, and Amplitude |
| **UTM Stripper** | Automatically removes tracking parameters (`utm_*`, `fbclid`, `gclid`, etc.) from URLs as you navigate |
| **Referrer Blocker** | Strips the `Referer` header so sites can't see which page you came from |
| **WebRTC Leak Blocker** | Prevents WebRTC from exposing your real IP address, even when using a VPN |
| **Canvas Fingerprint Blocker** | Adds invisible noise to canvas reads to defeat browser fingerprinting |
| **User-Agent Spoofer** | Disguises your browser as Chrome/Windows, Firefox, Safari, or a mobile device |
| **Cookie Banner Clicker** | Automatically dismisses cookie consent dialogs on page load |

### 🖥 Page

| Feature | What it does |
|---|---|
| **Autoplay Stopper** | Prevents videos and audio from playing automatically |
| **Force Dark Mode** | Applies a CSS dark mode to all websites |
| **Reader Mode** | Strips a page down to clean, readable text — click the button to activate on the current tab |

### 🧹 Clean

| Feature | What it does |
|---|---|
| **Sticky Element Killer** | Removes fixed headers, footers, and sidebars that follow you while scrolling |
| **Chat Widget Hider** | Hides chat bubbles and support widgets (Intercom, Drift, Zendesk, Crisp, HubSpot, Tidio, and more) |
| **Paywall Bypass Helper** | Removes CSS overlay and blur paywalls. Works on soft paywalls only — not server-side paywalls |
| **App Banner Remover** | Removes "Open in our app" banners on YouTube, Reddit, and other sites |

### ⚡ Productivity

| Feature | What it does |
|---|---|
| **Session Saver** | Save and restore groups of tabs by name |
| **Duplicate Tab Finder** | Finds and closes all duplicate tabs across all windows in one click |
| **Tab Limiter** | Warns you when your open tab count exceeds a limit you set |

### 🔧 Tools

| Feature | What it does |
|---|---|
| **Copy Tab URLs** | Copies the URLs of all open tabs (current window or all windows) to your clipboard as a plain list |
